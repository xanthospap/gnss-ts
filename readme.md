# GNSS-Derived Time Series Analysis Tools & Libraries

## Dependencies
    * [ggeodesy](https://github.com/xanthospap/ggeodesy)
      To install, just clone and compile.
      
      ```Shell
      git clone https://github.com/xanthospap/ggeodesy
      cd ggeodesy
      ./configure
      make
      sudo make install
      ```
